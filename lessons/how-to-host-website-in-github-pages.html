How to host website in github pages

GitHub Pages is a static site hosting service that takes HTML, CSS, and JavaScript files straight from a repository on GitHub, optionally runs the files through a build process, and publishes a website.


Before following the steps below you should create an account in Github https://github.com/:


  1. Login to your Github account
  2. On the upper right corner, click the + dropdown. Then, click New Repository
  3. Fill up the Repository name with this format [username].github.io (Ex: rynlbrn.github.io)
  4. Scroll down and click Create Repository
  5. Copy the url and open a terminal and type these in the terminal:
    • Navigate to Desktop by typing cd Desktop
    • Copy the url from the github repository and paste it on the terminal git clone <url>
    • Open a text editor and open the folder created earlier and create a file name index.html and type <h1>Sample HTML file</h1> inside the file then save
    • Back to the terminal, navigate to the folder by typing cd <foldername>
    • Type git init
    • Type git add index.html
    • Type git commit -m "first commit to github"
    • Type git remote add origin <url>
    • Type git push -u origin master, then open your github account
    • Back to github and reload the page and scroll down and look for Environments and click to github-pages
    • Finally, click View Deployment, and your website will appear (try it on your phone too..)
