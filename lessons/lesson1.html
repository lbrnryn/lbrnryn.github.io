How to combine strings

+ Operator VS Template String
      const firstname = "Juan";
      const lastname = "Dela Cruz";
      const age = 28;

      + Operator
      console.log("My name is " + firstname + lastname + "," + age + " years old")
      Output: My name is Juan Dela Cruz, 28 years old

      Template Literals / Template String
      console.log(`My name is ${firstname} ${lastname}, ${age} years old`)
      Output: My name is Juan Dela Cruz, 28 years old


      You will notice that it is easy to read the sentence using Template Literals / Template String than using the + Operator
