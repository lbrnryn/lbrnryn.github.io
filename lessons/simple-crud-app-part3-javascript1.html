Variables

Variables are containers for storing data or values. These are the rules when making a variable:

  • It can contains letters, numbers, underscores and dollar signs.
  • It must begin with a letter and can also begin with $ and _
  • It is case sensitive (name and Name are different variables)
  • Reserved words (Javascript keywords) cannot be used as variable
  • Can be written using lowercase or camelCase (these two are most used format when naming a variable)
  • Example:
    age = 30;
    lastName = "Doe";

Note: camelCase is used when there are 2 words, the first letter of the second word should be capitalize.


There are ways to declare a variable, two of them are let and const keyword that was introduced in ES6(2015). Here is the difference between let and const keyword:


  • let - keyword is used when the value of the variable will be change overtime. It is also use in a iteration like for loop.
    • Example:
      let score = 0;
      score = 1;
      for(let i = 0; i < 5; i++) { }
  • const - keyword is used when the value of the variable will not change.
    • Example:
      const firstName = "ross";
  • You will notice that I did not explain using var keyword because its an old way to declare a variable and I'm not going to use that in this tutorial.
