Heading Tag

Ang heading tag ay ginagamit bilang mga title or subtitle sa isang website. h1 to h6 (Biggest to smallest font of heading)

HTML OUTPUT
<h1>This is heading 1</h1>

This is heading 1
<h2>This is heading 2</h2>

This is heading 2
<h3>This is heading 3</h3>

This is heading 3
<h4>This is heading 4</h4>

This is heading 4
<h5>This is heading 5</h5>
This is heading 5
<h6>This is heading 6</h6>
This is heading 6

Form Tag

Ang form tag ay ginagamit para makakolekta ng mga input ng mga user ng isang website or app at kadalasang ipinoproseso ito sa backend katulad ng Node JS, PHP, Python at iba pa.


<form></form>

At sa mismong start tag <form> ay merong inilalagay na mga attributes katulad ng action(routes kung saan dadalhin ang mga nakolektang input galing sa user) at method(GET or POST(ito ang kadalasan ginagamit kaysa sa GET)) attributes

<form action="/sampleroute" method="POST"></form>

Form Element (label)

Ang <label> tag ay ginagamit para magbigay kahulugan sa isang input box upang malaman ng mga user kung ano ang ilalagay nila dito. Meron itong for attribute na kung saan dapat pareho ito ng id ng input tag na tinutukoy nito.

HTML OUTPUT
<label for="myText">Click here to show the blinking text cursor on the input</label>
<input type="text" id="myText">

Ang blinking text cursor ay ang vertical line na makikita kapag pinindot ang isang input element



Form Element (input type="text")

Ang input type="text" ay isa sa mga klase ng input na kadalasang ginagamit sa form element, sa pamamagitan nito makukuha natin ang mga informasyon na ilalagay ng user sa form element na ito

HTML OUTPUT
<input type="text">
