Form

Form tag is used to collect user inputs and frequently processed in the backend like Node JS, PHP, Python and other backend languages.


<form></form>


Form has attributes like action(routes where user input goes) and method post


<form action="/sampleroute" method="POST"></form>



Form Element (label)

Label tag is used to give input element meaning. It has a for attribute with a value that is the same as the id name of the input its referring to, so the hidden javascript on this element will work.

HTML Tag OUTPUT
<label for="myText">First Name</label>
<input type="text" id="myText">

Blinking text cursor is the vertical line that blinks in and out, so when you click the label element, the blinking text cursor on the input its referring to will appear.



Form Element (input type="text")

Input type="text" is one of the type of input element frequently used in a form, this will get any information or data that is being type in here by the user.

HTML Tag OUTPUT
<input type="text">


Button (button type="submit")

Button type="submit" is one of the 3 types (submit | reset | button) of button which submits the current form data. It is similar to input type="submit" in terms of submitting the data from the form to the client or backend

HTML Tag OUTPUT
<button type="submit">Submit<button>


UL(Unordered List)

UL is one of the 2 types (ul | ol) of list that also groups a set of related items and its list of items will be marked with bullets by default. There are also different list item markers such as (disc | circle | square | none) in Unordered List and (type = "1" | "A" | "a" | "I" | "i") in Ordered List.

HTML Tag OUTPUT

<ul>

  <li>List Item</li>

</ul>

Coding Time

Now we know how to use those HTML elements, so we are going to use what we learn to make the basic structure of our Simple CRUD App. Follow these steps:


  1. On the Desktop, create a folder named CRUD APP
  2. Then, open your text editor, click Add Folders on the left (If you use Atom)
  3. Then, locate and select the folder you created and click Select Folder
  4. You will notice on the left part of the text editor the folder you select
  5. Right click on the folder, and click New File and name it as index.html
  6. In the index.html, type ! (exclamation point) then press tab, you don't need to type those first codes anymore, it's possible with emmet plugin/tool that we download earlier.
  7. On the <body></body> tag:
    • Type h1, then press tab, then press Enter
    • Type form>label+input#name+button:submit, then press tab, then press Enter
    • Type ul, then press tab, then press Enter
    • Type script:src, then press tab
  8. Finally, we need to put these text in some html tags:
    • Inside the <title></title> write Simple CRUD App (this is optional)
    • Inside <h1></h1> write My Friends
    • In the for attribute inside the <label> start tag, write name
    • Inside <label for="name"></label> write Name
    • Inside <button></button> write Submit
    • In the src attribute inside the <script> start tag, write index.js
