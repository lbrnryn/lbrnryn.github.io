Simple CRUD Application (Part 1 - Introduction and Setup)

This tutorial will give you an idea on how to create an app using CRUD. CRUD stands for Create, Read, Update and Delete. To understand this better, we will take Facebook as an example:


  • Create - Users can register for an account, post text/images/videos, write comments/chat messages.
  • Read - Users can view their own account and other user's post, images, videos, comments, chat messages.
  • Update - Users can edit their own account, post, images, videos, comments, chat messages.
  • Delete - Users can remove their own account, post, images, videos, comments, chat messages.

Let's get started!

You need to prepare text editor that you will use to get along in this tutorial. A text editor is a software that allows you to write code on your computer. Here are some of the list of popular text editors and their website where you can download them:


  • Atom - http://atom.io
  • Visual Studio Code - https://code.visualstudio.com/
  • Brackets - https://brackets.io/
  • Sublime Text - https://www.sublimetext.com/

So, in this tutorial I will use Atom as my text editor, you need to download a plugin/tool named emmet which will help us to code faster and atom-live-server which will reload the browser automatically when we change something in our HTML. Follow these steps to download a plugin/tool in your text editor:


  1. Open you text editor
  2. On the top left corner, click File
  3. Then click Settings, settings tab will appear on the screen, type emmet
  4. Next, scroll down and look for emmet (2.4.3 with 4.8+ millon downloads), click Install
  5. Next, type atom-live-server (2.3.0 with 1.4+ million downloads), click Install
  6. Finally, close the settings tab


On the part 2 of this series, we will learn how emmet and atom-live-server helps us coding faster and better.
