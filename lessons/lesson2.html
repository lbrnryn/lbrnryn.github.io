Simple CRUD Application (Part 1 - Introduction and Setup)

Ang tutorial na ito ay magbibigay sa inyo ng ideya kung paano gumawa ng isang system or application gamit ang CRUD. CRUD stands for Create, Read, Update, Delete. I-halimbawa natin ang facebook Create - nakakapagpost tayo ng mga mensahe, imahe, video. Read - nakikita natin ang mga naipost natin na mensahe, image, o video kasama na rin ang mga mensahe, imahe o video ng ibang user Update - nababago natin ang mga mensahe, image, o video na naunang ipinost. Delete - nabubura natin ang lahat ng iyon Sa tutorial na ito makikita kung paano gumawa ng simpleng CRUD Application na laging nagagamit sa paggawa ng mga application. Bago tumuloy sa Part 2 ng tutorial na ito ay kailangan ihanda ang text editor maaring gumamit ng notepad, visual studio code, o kahit anong gusto mo pero ang gagamitin ko dito ay ang ATOM text editor. Maari ninyong idownload ang atom sa link na ito Atom kung gusto ninyong gamitin ito. Kapag nadownload ninyo na ang ATOM o ang kahit anong text editor, buksan mo ang text editor at i-click ang File and then click Settings, next click + Install at i-type ang atom-live-server at makikita ninyo ang 1M + na downloads and click install at i-type din ang emmet at makikita ninyo ang 4.8M + na downloads and click install at pagkatapos i-close na ang settings tab at i-click ang ADD FOLDERS at magcreate ng folder and then open yung folder na ginawa mo. Try mo magcreate ng file, sample index.html and then i-type mo and exclamation point na ! then press tab at lalabas ang Boiler Plate code ng HTML Go to Part 2
